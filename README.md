6. Update progress file before moving to next topic

- Prefer scenario-based questions over theory-only study
- Keep notes quiz-mapped and exam-oriented
- Keep each topic self-contained (no mandatory external dependency)
- Add optional citations only for deep dives
- Track all outcomes in `progress.md`
